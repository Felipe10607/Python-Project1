# Python-Project1

A proposta é do projeto foi aplicar orientação de objetos e utilizar a arquitetura MVC(Model - View - Controller) em prática. Portanto esse projeto simula um sistema de gerenciamento de uma loja. 

Planejar e colocar em escopo o que cada parte da arquitetura faz que é um pouco mais complicada. Hora ou outra você volta para adicionar ou função nova ou cadastra mais um detalhe. Acredito que devo tentar planejar e preparar melhor antes de sair codando o que vai na cabeça.
